# TP5A - Securite avec Podman

## Objectifs
- Maitriser la securite rootless
- Configurer SELinux/AppArmor
- Scanner les vulnerabilites
- Gerer les registres securises

## Contenu

- `scripts/` - Scripts de securite
- `exemples/` - Exemples de configuration

## Principes de securite

1. Utiliser rootless quand possible
2. Limiter les capabilities
3. Scanner les images regulierement
4. Utiliser des utilisateurs non-root
